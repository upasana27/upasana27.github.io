# Academic Profile Website - Template

A professional academic webpage template for PhD students and researchers.

## Setup Instructions

1. **Add your profile photo**: Replace `profile-photo.jpg` with your actual photo
2. **Update personal information** in `index.html`:
   - Your name (already set to "Upasana Biswas")
   - University name and department
   - Research area/field
   - Email address
   - GitHub username (if applicable)

3. **Fill in content sections**:
   - **About**: Your bio, background, and achievements
   - **Education**: Degree details, advisor name, thesis topic
   - **Research Interests**: Your specific research areas
   - **Projects**: Current and past research projects
   - **Publications**: Add your papers, conferences, and preprints
   - **Teaching**: TA experience and other teaching roles
   - **Contact**: Office location and contact details

## Customization

### Colors
Edit CSS variables in `styles.css` to change the color scheme:
```css
:root {
    --primary-color: #2c3e50;
    --secondary-color: #3498db;
    --accent-color: #e74c3c;
}
```

### Layout
The website is fully responsive and works on mobile, tablet, and desktop devices.

## How to Use

1. Open `index.html` in a web browser to preview
2. Replace all `[placeholder text]` with your actual information
3. Add your profile photo as `profile-photo.jpg` in the same folder
4. Upload to your preferred hosting service (GitHub Pages, personal server, university hosting, etc.)

## Hosting Options

- **GitHub Pages**: Free hosting for static sites
- **Netlify**: Free tier with easy deployment
- **University Server**: Many universities provide web space for students
- **Personal Domain**: Purchase a domain and use any hosting service
